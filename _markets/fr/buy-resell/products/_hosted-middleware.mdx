### ChaîneCloud

*ChaîneCloud* is fiskaltrust's SaaS Middleware platform.

It offers the same set of features as a locally hosted Middleware instance, but without the need to install or maintain any software on your side. All your POS system needs is a working Internet connection and it can communicate with *ChaîneCloud* over an encrypted, **HTTPS-secured** connection.
