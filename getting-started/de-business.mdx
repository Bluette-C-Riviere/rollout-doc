|             Step             | Description                                                  |
| :--------------------------: | ------------------------------------------------------------ |
| ![](../images/numbers/1.png) | Open your the rollout management plan page from `Rollout Management` / `Plan` in the left-hand navigation menu |
| ![](../images/numbers/2.png) | Select `Business Rollout - Rolling out products`             |



![](../_markets/de/getting-started/my-first-cashbox/images/biz/02_select-product.png)

|             Step             | Description                                      |
| :--------------------------: | ------------------------------------------------ |
| ![](../images/numbers/3.png) | Filter for `fiskaly` in the search field         |
| ![](../images/numbers/4.png) | Limit the entries to `Dealerpackage based plans` |
| ![](../images/numbers/5.png) | Select plan **4445-0006**                        |



![](../_markets/de/getting-started/my-first-cashbox/images/biz/03_select-operator.png)

|             Step             | Description                                          |
| :--------------------------: | ---------------------------------------------------- |
| ![](../images/numbers/6.png) | Select `Perform the rollout for the current account` |



![](../_markets/de/getting-started/my-first-cashbox/images/biz/04_select-outlet.png)

|             Step             | Description                                                  |
| :--------------------------: | ------------------------------------------------------------ |
| ![](../images/numbers/7.png) | Tick the desired outlet where you would like to roll out the Middleware |
| ![](../images/numbers/8.png) | Click `Submit Selection`                                     |



![](../_markets/de/getting-started/my-first-cashbox/images/biz/05_create-quote.png)

|             Step             | Description          |
| :--------------------------: | -------------------- |
| ![](../images/numbers/9.png) | Click `Create quote` |

Creating the quote can take a few seconds. Once it is ready, the button will change to `Load quote into cart`. Click the button again to place the quote into your shopping cart. This will redirect you to the shopping cart page.



![](../_markets/de/getting-started/my-first-cashbox/images/biz/06_shopping-cart.png)

|             Step              | Description                                                  |
| :---------------------------: | ------------------------------------------------------------ |
| ![](../images/numbers/10.png) | Verify that the items in your shopping cart match the selected products<br /><br />\* 1 TSE-as-a-Service fiskaltrust.Sorglos entitlement<br />\* 1 entitlement-based fiskaly TSE<br />\* 1 entitlement-based fiskaltrust.Sorglos bundle |
| ![](../images/numbers/11.png) | Click `Checkout`                                             |



![](../_markets/de/getting-started/my-first-cashbox/images/biz/07_checkout.png)

|             Step              | Description                                 |
| :---------------------------: | ------------------------------------------- |
| ![](../images/numbers/12.png) | Click `binding order` to complete the order |



At this point should have the order listed under `Shop` / `Orders`. Finishing the order can take a couple of minutes, but once it has been  completed there should be the following items active in your account

- two **demo subscriptions**, for Carefree and TSE-as-a-Service, under `Shop` / `Subscriptions`
- a **fiskaly TSE** for the selected outlet under `Configuration` / `Signature Creation Unit`
